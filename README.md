# Meu Projeto de Visão Computacional

Este é um projeto simples de Visão Computacional utilizando OpenCV. O objetivo é aplicar detecção de bordas em imagens e capturar vídeo da webcam em tempo real.

## Como Configurar o Projeto

### Requisitos

Certifique-se de ter o Python instalado (versão 3.6 ou superior). Você pode instalar os pacotes necessários com o comando:

```bash
pip install -r requirements.txt
